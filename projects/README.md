# Your projects

This is where **you** build. Each automation you create is its own **project** — a
self-contained folder under here.

For Exercises 2, 3, and 4 you'll create a new folder here, e.g.:

```
projects/
├── morning-briefing/        ← Exercise 2
├── combined-assistant/      ← Exercise 3
└── <your-automation>/       ← Exercise 4
```

A simple project usually has:
- the **process** you designed in Cowork (the instructions/prompt you reuse),
- a **storage** folder for its data and outputs,
- any **config** it needs (e.g., topics, your email address).

You don't need to design anything fancy — start a folder, grant Cowork access to it,
and build. Exercise 1 (in `exercises/01-inbox-digest/`) is a worked example you can
look at for the shape of a project.
